# Roadmap

## Phase 1: Core Platform

- [x] Go backend API
- [x] Vue control plane UI
- [x] Project registry
- [x] Service scaffolding
- [x] Release workflow
- [x] Service-scoped deployments
- [x] Dockerized local environment

## Phase 2: Developer Experience

- [ ] Deployment logs and runner output in UI
- [ ] Better validation and guided forms for scaffold variables
- [ ] Service health and runtime status panels
- [ ] Stronger auth and SSO flows
- [ ] More plugin management controls from the UI

## Phase 3: Delivery Automation

- [ ] Richer GitOps status and sync visibility
- [ ] Automatic rollback handling
- [ ] Progressive rollout workflows
- [ ] Multi-cluster or multi-target deployment support

## Phase 4: Platform Insights

- [ ] Usage analytics by team and project
- [ ] Notifications for scaffold, release, and deployment events
- [ ] Metrics dashboards and operational insights
- [ ] Audit and compliance reporting

## Future Ideas

- [ ] AI-assisted scaffold suggestions
- [ ] Secret manager integration
- [ ] Feature flag management
- [ ] More scaffolder and deployer templates
